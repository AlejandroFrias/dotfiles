This management command regenerates (deterministically) the gold reports for
the report regression tests. It generates html files for each of the
fake_results scenarios in REPORT_HTML_DIR. As this is a computation heavy
management command and can take a couple hours to regenerate all of them,
there is a pipeline where you can push your branch/commit and it will generate
all of the golden files. Instructions for how to use the pipeline are here:
http://discourse.counsyl.com/t/how-do-i-create-the-golden-files-using-the-
test-report-data-pipeline/601/2. # nopep8 These files have relative rather
than absolute paths to the images to reduce diffs when the reports need to be
updated. Because of this our python wrapper for prince, prince_pdf() will not
correctly generate pdfs from these files. Simply run prince from the command
line: prince REPORT_HTML_DIR/SCENARIO_NAME.html
